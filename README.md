# HTMLMaps

HTMLMaps is a Next.js landing page for validating demand for a no-code builder that creates clickable HTML/SVG maps, floor plans, image hotspots, seating charts, venue layouts, campus maps, sales territory maps, real estate site plans, and interactive visual embeds for websites.

The project is planned to be hosted at:

**https://www.htmlmaps.com**

## Purpose

Many websites still handle maps, floor plans, site plans, and diagrams as flat images. That works for basic display, but it does not help users click a room, open a location page, view a tooltip, inspect a section, or move from a visual layout to the next step.

HTMLMaps is designed to test demand for a simpler solution:

> Create clickable maps and visual layouts for websites without writing custom code.

The intended product direction is a hosted embed-first builder where users can create an interactive visual, publish it, and paste a responsive iframe embed into their website.

## Planned Product Model

The planned product model is intentionally lightweight:

1. User chooses a built-in map template or uploads a visual.
2. User adds clickable regions, hotspots, labels, tooltips, colors, and links.
3. User publishes the project as a hosted iframe embed.
4. User receives a private edit link.
5. The embed can be placed on common website platforms that support custom HTML or iframe embeds.

A future paid model may include:

- 30-day unbranded hosted trial
- Small HTMLMaps badge after the trial
- One-time paid unlock per project
- Commercial-use license
- Optional downloadable HTML/CSS/JS/SVG export package
- Custom map and floor plan setup services
